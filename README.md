## RUNNING THE GAME FROM THE SOURCE FILES

Windows users
```
java -Dfile.encoding=UTF-8 -classpath "lib\junit-4.13.2.jar;out\production\CE812_lowe_rachel_m" crappyGame.GameRunner
```

Linux users
```
java -Dfile.encoding=UTF-8 -classpath "lib\junit-4.13.2.jar:out\production\CE812_lowe_rachel_m" crappyGame.GameRunner
```
